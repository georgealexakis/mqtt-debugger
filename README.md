# MQTT Debugger

 MQTT Debugger is a single HTML page tool for debugging of [IoT Agent Application](https://georgealexakis.github.io/iot-agent/) MQTT connection.

## Technologies

 * Paho MQTT
 * Bootstrap
 * Javascript
